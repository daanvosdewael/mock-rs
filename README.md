# MoCk

[![CI](https://github.com/daanvosdewael/mock-rs/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/daanvosdewael/mock-rs/actions/workflows/ci.yml)

BeCAuSe SoMeTiMeS YoU NeeD To MaKe a PoiNt

A CLI tool that converts your text into mocking case. You know, *that* alternating caps thing. Features a vowel-weighted algorithm that makes it look just right and an optional garble mode for when you really want to commit.

</div>

## iNsTAlLATioN

```sh
# To BeH iMPleMeHnTeD...
```

## USaGe

```sh
$ mock "hello world"
HeLlo WoRlD

$ echo "hello world" | mock
HeLlo WoRlD
```

## GaRBle MoDeH

FoR wHeHn ReGcHUlAR MoCKiHnGh iShJ NoT eHNoUWgH.

Garble mode applies phonetic replacements before mocking, producing output that is aggressively unreadable. Ideal for emphasizing just how little you agree with someone.

```sh
$ mock --garble kompjoeteren
KoHMmPJoeHdTeReHn

$ mock --garble gezellig
GeHZSeehLLiGcH
```
